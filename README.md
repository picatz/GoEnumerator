# GoEnumerator
A tool wrote in GO for very fast pasive enumeration of a target

---
Build for GNU/Linux  
- make

Build for Darwin
- make darwin

Build for Windows
- make windows

Run:  
- GoEnumerator hispagatos.org
- do not add http/https in front.
